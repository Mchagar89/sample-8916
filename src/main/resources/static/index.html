OpenRedukti + Spring Boot

Interest-rate derivatives pricing via a JNI bridge to the OpenRedukti C++ library.

Endpoints

GET /api/hello – health ping

GET /api/redukti/version – library version

POST /api/redukti/bootstrap – bootstrap zero curves from par rates

POST /api/redukti/value – compute IRS present value & DV01
